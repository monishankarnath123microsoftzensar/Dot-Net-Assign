2.Difference & similarity between int.parse and convert.ToInt32()
Ans: Both int.Parse and Convert.ToInt32 are used to convert string into the integer but Only difference between them is to 
Convert.ToInt32 handle null and returns '0' as output and
int.parse is not going to handle NULL and will give a Argument Null Exception.
Passing a null value to int Parse will throw an ArgumentNullException while passing a null value to Convert ToInt32 will give zero.
